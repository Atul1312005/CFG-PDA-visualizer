# CFG ↔ PDA Visualizer

An interactive web-based tool to convert between **Context-Free Grammars (CFG)** and **Pushdown Automata (PDA)** with step-by-step visualization.

## Features

*  CFG → PDA conversion using formal construction rules
*  PDA → CFG conversion using standard algorithm
*  Step-by-step transformation breakdown
*  Stack simulation with animation
*  Interactive PDA state diagram (D3.js)
*  Built-in theory section for learning


##  Tech Stack

* HTML, CSS, JavaScript
* D3.js (for graph visualization)
* Automata Theory algorithms

##  How to Run

1. Clone the repository:

   ```bash
   git clone https://atul1312005.github.io/CFG-PDA-visualizer/
   ```

2. Open:

   ```
   index.html
   ```

---

## Example

**Input CFG:**

```
S → aSb | ε
```

**Output:**

* PDA transitions
* Stack simulation
* State diagram
